# 📊 Data Professional Survey Dashboard

An interactive Power BI dashboard analyzing survey responses from **630 data professionals** worldwide.

## 📁 Dataset
- **Source:** Alex The Analyst - Data Professional Survey
- **Respondents:** 630 survey takers
- **Average Age:** 29.87 years

## 📌 Dashboard Highlights

- **Average Salary by Job Title** — Data Scientists lead with the highest average salary
- **Country of Survey Takers** — USA, India, UK, Canada, and others
- **Favorite Programming Languages** — Python dominates across all roles
- **Average Salary by Gender** — Male: $55.37K | Female: $53.61K
- **Work/Life Balance Happiness** — Average score: 5.74 / 10
- **Salary Happiness** — Average score: 4.27 / 10

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (data cleaning & transformation)
